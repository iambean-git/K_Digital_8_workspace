# 파이썬 머신러닝 수업

### 기본 설치 (권장)
```
pip install pandas scikit-learn seaborn graphviz
```

### pip 관련 명령어
- 설치된 리스트 확인 ````pip list```
- 설치 (예, pandas 설치) : ```pip install pandas```
- 버전 지정 설치 : ```pip install numpy==2.0.2```
- 설치 및 업데이트 : ```pip install -U pandas``` (공부할때만 권장)
- 삭제 ```pip uninstall pandas```
